Техническое задание

Приложение Калькулятор

1. Вычисление формул, содержащих математические выражения, состоящие из:
    • знаки арифметических операций сложения, вычитания, умножения и деления
    • скобки для определения последовательности операций
2. Результат выводится под введенной формулой
3. Ввод значений осуществляется с помощью отдельной отрисованной клавиатуры
4. Итогом разработки должно явиться приложение под ОС Android в виде apk-файла.
5. Промежуточные итоги разработки хранить в github
